## Базовые команды

```bash
git init               # Инициализация нового репозитория
git clone <url>        # Клонировать репозиторий
git status             # Статус файлов
git add .              # Добавить все файлы
git commit -m "Name"   # Коммит
git log --oneline      # Краткий лог
